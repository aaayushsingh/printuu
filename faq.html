Frequently Asked Questions


Q1-How should I use printuu?

Ans- Just follow these simple steps-

  • Log on to printuu.com and upload your desired file
  • You will be given a 4 digit token number
  • Use this token number at any printer shop(or any printer) to get your printouts


    • Q2- How should I use this token number?

    Ans- Use this token number at any printer shop(or any printer) to generate your print outs in seconds.
    Make sure that you have unblocked the popups before entering the token number.


    Q3- Where should I enter the token number?

    Ans- On printuu.com, there is a separate print tab where you can enter your token number Follow this link


    Q4- Why should I use Printuu?

    Ans- With printuu-

  • No more queues for getting your print outs. Upload any file from anywhere and print in seconds.
  • No more worrying about accessing your E-mail at unsafe public computers.
  • Share all your assignment and projects hassle free.


    • Q5-For how long is the token number valid?

    Ans- The token number is valid for 2 days. You can use this token as many number of times as you want.


    Q6- What if I forget my token number?

    Ans- Don't worry. An automated mail with the token number is sent to your registered e-mail id when you upload your document.


    Q7- What all file types are accepted?

    Ans- We accept all types of text/image files. But make sure that the file size is less than 20 MB.
